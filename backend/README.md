# Mechanical Portfolio Backend

Backend API cho website portfolio cá nhân lĩnh vực Cơ khí & Điện.

## 🚀 Công nghệ sử dụng

- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **Cloudinary** - File storage
- **Multer** - File upload

## 📋 Yêu cầu hệ thống

- Node.js >= 14.x
- MongoDB >= 5.x
- npm hoặc yarn
